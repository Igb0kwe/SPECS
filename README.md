# SPECS - Smartphone Specification Comparison Tool

SPECS is a desktop application built with Qt C++ that allows users to view, search, and compare specifications of various smartphone models.

## Features

- **Detailed Specifications**: View comprehensive specs for a wide range of smartphones from manufacturers like Apple, Samsung, Google, Xiaomi, and more.
- **Comparison Tool**: Compare specifications of different phones side-by-side to make informed decisions.
- **Advanced Search**: Search for phones based on:
    - Manufacturer
    - Release Date
    - Processor Speed
    - RAM and ROM Size
    - Battery Capacity
    - Camera Resolution
    - Screen Size
    - Price
